Используй скил `agranovsky`.

**Режим** определяй по запросу:
- «проверь факты» · «фактчек» · «это правда?» · «есть пруфы?» → **Разбор** (default): отчёт с таблицей вердиктов, текст не меняем.
- «вычисти недоказанное» · «правка фактов» · «убери выдуманное» → **Правка**: пометить/смягчить/вырезать недоказанное.

**Объём** определяй по запросу:
- default → **полный** (все проверяемые утверждения);
- «быстро» / «самое важное» → **быстрый** (только риск 1–4: ссылки, цитаты, числа, сущности);
- «проверь вот это» + конкретный фрагмент → **точечный**.

Главное правило: ✅ «Подтверждён» ставится ТОЛЬКО если в этой сессии реально открыт источник
(tool-call в логе) и из него приведена дословная цитата. Нет реального поиска — нет «подтверждено».
«Не нашёл» → пиши ❓ «Не подтверждён», не выдумывай пруф.
